# 🪨 Minecraft PE Addon：石像系列生物擴充包

本專案為我自製的 Minecraft PE 生物 Addon，主題圍繞古老遺跡與石像魔物，加入了多種全新敵對與中立生物，擴展探險與戰鬥體驗！

## 🧩 特點 Features

- 完整支援 Minecraft Bedrock / PE
- 高度還原 Minecraft 方塊比例與貼圖風格
- 每個生物皆搭配獨立技能與掉落
- 包含 Boss 生物與可互動的友善單位

---

## 🔹 生物介紹 Creatures Overview

### 🪨 石像守衛（Stone Guardian）
- **類型**：中立 / 近戰
- **特徵**：只會在遺跡附近或玩家開啟寶箱時激活，會使用強力重擊與震地範圍攻擊。
- **掉落物**：古老石塊、符文碎片、守護者之心

---

### 👁️ 寂衛（Dungeon Watcher）
- **類型**：敵對 / 遠程
- **特徵**：漂浮於空中，能發射能量束；平時幾乎無法察覺，靠近後才會醒來。
- **掉落物**：觀察之眼、符文結晶、封印碎片

---

### 🔗 鏈錘獸（Chain Golem）
- **類型**：敵對 / 中距離坦克型
- **特徵**：
  - 肌肉壯碩，右臂為長鍊連接石錘，具中距離攻擊與拉扯技能。
  - 生命值減半後會釋放熔岩光環與粒子爆發。
- **技能**：
  - 長鏈猛擊（暈眩與地形破壞）
  - 熔岩爆裂（釋放火焰光環）
  - 拉拽（將玩家拉至身邊）
- **掉落物**：鍊鋼碎片、熔岩之心、重擊核心

---

### 🪨 碎石精靈（Rubble Sprite）
- **類型**：友好 / 支援型
- **特徵**：
  - 小型碎石精靈，可撿拾掉落物與與玩家互動（餵食石頭可兌換礦物）。
  - 遇敵會逃跑或偽裝成地形方塊。
- **掉落物**：雜礦碎片、小礦物、小型背包

---

### 🏛️ 遺跡管理者（Warden of the Ruins）
- **類型**：Boss
- **特徵**：
  - 高瘦石像 Boss，持有雙刃斧，出現在遺跡最深處。
  - 具有多段式攻擊與召喚守衛靈魂的能力。
- **掉落物**：鎧甲碎片、符文能量、水晶之核

---

## 🗂️ 安裝說明 Installation

1. 將下載的 `.mcaddon` 檔案導入 Minecraft PE。
2. 啟用資源包與行為包。
3. 建議於創造模式先行測試生物生成與功能。
4. 可使用 `/summon` 指令手動生成各生物。
